# Flex
### 父级别常用属性
1. flex-direction:设置主轴的方向
   1. row 默认值是从左到右
   2. row-reverse 从右到左
   3. column 从上到下
   4. column-reverse 从下到上

2. justify-content 设置主轴的子元素排列方式
   1. flex-start:默认值从头部开始，如果主轴是X，则从左到右。
   2. flex-end 从尾部开始排列
   3. center 在主轴居中对齐（如果主轴是X轴，则水平居中）
   4. space-around 平分剩余空间
   5. space-between 先两边贴边，再平分剩余空间
